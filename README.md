# Wide #

## Intro ##
A simple <b>W</b>eb **IDE** for golang.

## Features ##

* Code Highlight
* Autocomplete
* Format
* Run & Debug
* Multiplayer
* Navigation & Jump
* Shell
* Git integration
* Web development

## Demos ##
* [20140823](http://b3log.org/wide/demo/20140823.html)

## Setup ##

1. Downloads source
2. Compiles wide with `go build` 
3. Configures `conf/wide.json`
4. Runs the executable `wide` or `wide.exe`
