搭建服务器

基础配置

  1. 下载 Wide 安装包
  2. 解压到安装路径 {wide}，例如 /root/wide
  3. 配置 {wide}/conf/wide.json（可选）
现在已经可以使用用户 admin，密码 admin 登录使用了。

多用户配置

前提：需要安装 git。
  1. 初始化主库：在 {wide}/data/workspace/src 下执行 git init
  2. 添加用户：修改 {wide}/conf/wide.json 中的 Users
  3. 添加用户库：在 {wide}/data/user_workspaces/{user}/ 中 git clone 主库
